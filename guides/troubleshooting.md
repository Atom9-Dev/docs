# Troubleshooting

## Common errors
- `app_slug_required` -> provide app slug in request.
- `auth_app_not_found` -> create app via onboarding first.
- `auth_method_not_enabled_for_app` -> enable method in app config.
- `callback_not_allowed` -> add callback to app allowlist.
- `sensitive_auth_required` -> use stronger auth context.
- `rate_limited` -> retry after window.
